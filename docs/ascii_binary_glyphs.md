## Pattern Notes

- **Lowercase (01100xxx‚Äď01111xxx):** All start with `011`
- **Uppercase (01000xxx‚Äď01011xxx):** All start with `010`
- **Digits (00110xxx‚Äď00111xxx):** All start with `0011`
- Bit 5 (second bit from left) distinguishes uppercase from lowercase: `0` = uppercase, `1` = lowercase
- Each 8-bit byte renders as a 3√ó3 grid with 1 unused cell (layout: 3 rows √ó 3 columns = 9 cells, but we use 8 bits; the 9th cell is always `‚ĖĎ`)
