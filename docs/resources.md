# Marain Resources

## Implementations and tools

- [zakalwe2040/marain](https://github.com/zakalwe2040/marain) — Tonal Marain: a 24-consonant abjad, numeral system, and operator notation, rendered as SVG glyph diagrams; uses a 4×5 lattice extending the M1 3×3 grid

## Fonts
- [tomdionysus/marain-font](https://github.com/tomdionysus/marain-font) — TrueType font (`Marain.ttf`) implementing Banks' phoneme alphabet; glyph outlines are a source of truth for precise binary values

## Translation
- [New Marian Translator](https://lingojam.com/NewMarianTranslator\(Updated\)) — Replaces common english words with their Marian counterparts in English
- [Marian Tools](https://marain-tools.netlify.app/?SC) - Translates English words into their Marian phonetic glyphs. Also includes an English to Marain dictionary
